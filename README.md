# TestTaskDeribitClient
Внешнее API для обработки сохраненных данных на FastAPI
Для сохранения данных написан клиент, который обрщается на сайт криптобиржы Deribit каждую минуту забирает цену btc_usd и eth_usd (index price валюты) после чего сохраняет в базу данных (PostgreSQL) тикер валюты, текущую цену и время в UNIX timestamp.
